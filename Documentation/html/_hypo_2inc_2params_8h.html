struct paramStorage_struct paramStorage[]

This large array contains all the tuning parameter defaults.

Tuning parameters are stored in EEPROM, this array contains the defaults, which are overwritten by the EEPROM values on startup if the EEPROM checksum is valid.

unsigned char paramWaitForRemote

Boolean for determining whether to wait for the remote device to transmit its parameters
